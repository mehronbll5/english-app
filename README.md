# 🎓 English Dars

Образовательная веб-платформа для изучения английского языка,
разработанная специально для таджикских студентов.

## ✨ Функции
- 🎬 Структурированные видеоуроки (уровни A1–C1)
- 📝 Интерактивные квизы после каждого урока
- 📚 Трёхъязычный словарь (English / Тоҷикӣ / Русский)
- 📈 Персональный трекер прогресса и достижений

## 🛠 Технологии
- React
- Tailwind CSS
- Vite
